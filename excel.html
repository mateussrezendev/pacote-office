Excel

O Microsoft Excel é um software de planilha eletrônica amplamente utilizado para análise e organização de dados em formato de tabelas. Ele oferece uma ampla gama de recursos, incluindo cálculos avançados, gráficos e ferramentas de formatação.

Atalhos do Excel

Atalho Função
Ctrl + N Aplicar ou remover negrito no texto
Ctrl + I Aplicar ou remover itálico ao texto
Ctrl + S Sublinhar o texto (ou remover sublinhado)
Ctrl + A Abrir caixa de diálogo “Argumentos da Função”
Ctrl + End Mover para a última célula na planilha que contém dados
Ctrl + Home Mover para a primeira célula na planilha
End Ativar o Modo de Término
Home Mover para o início da linha
Ctrl + Shift + End Selecionar intervalo até a última célula no canto inferior direito
Shift + Tab Alternar para célula anterior ou opção anterior na caixa de diálogo
