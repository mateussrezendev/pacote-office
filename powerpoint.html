PowerPoint

O PowerPoint é um software desenvolvido pela Microsoft que permite criar apresentações visuais de forma simples e dinâmica, com a possibilidade de incluir textos, imagens, gráficos, vídeos e efeitos especiais. É amplamente utilizado em ambientes corporativos, educacionais e de entretenimento.

Atalhos do PowerPoint

Atalho Função
Ctrl + N Colocar o texto selecionado em negrito
Ctrl + M Adicionar um novo slide
Ctrl + D Duplicar um objeto
Ctrl + Z Desfazer a última ação
Ctrl + Y Refazer a última ação
Ctrl + O Criar nova apresentação
Ctrl + C Copiar o texto, objeto ou slide selecionado
Ctrl + V Colar o texto, objeto ou slide recortado ou copiado
Ctrl + X Recortar o texto, objeto ou slide selecionado
F5 Iniciar a apresentação de slides

Recursos do PowerPoint

  • Criar apresentações
  • Adicionar texto, imagens, arte e vídeos
  • Escolher um design profissional com o Designer do PowerPoint
  • Adicione transições, animações e movimento cinematográfico
  • Salvar no OneDrive para acessar suas apresentações no computador, tablet ou telefone.
