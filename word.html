Microsoft Word

O Microsoft Word é um processador de texto desenvolvido pela Microsoft. Ele permite criar, editar e formatar documentos de texto de forma rápida e fácil.

Atalhos do Microsoft Word

Atalho Descrição
Ctrl + N Formata o texto para negrito
Ctrl + O Abre um novo documento em branco
Ctrl + S Sublinha o texto selecionado
Ctrl + P Imprime seu documento
Ctrl + Z Desfaz a última ação
Ctrl + R Refaz a última ação
Ctrl + A Abre um arquivo já existente
Ctrl + B Salva o documento
Ctrl + I Formata o texto para itálico
Ctrl + U Substitui texto, formatação ou outros itens

Recursos do Microsoft Word

  • Criar e editar documentos de texto
  • Formatar texto, como fonte, tamanho e cor
  • Inserir imagens, tabelas e gráficos
  • Criar e aplicar estilos de parágrafo e de caractere
  • Criar e utilizar modelos de documento
  • Corrigir ortografia e gramática
  • Inserir cabeçalhos e rodapés
  • Criar índices e sumários
  • Colaborar em tempo real com outras pessoas em um mesmo documento
